This site is a webhook proxy for CircleCI. It will only allow webhooks matching rules all others will be silently ignored.

Forked PR Filter

  • Only filters Forked PRS, pushes and same-repo PRS all build as normal
  • Forked PRs are checked for the label provided in URL

Replace your webhook with 

http://buildbully.cfapps.io/check_for_label/LABEL
